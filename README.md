HOTEL QUINTA DALAM

Proyecto desarrollado por Eduardo Lopez Martinez y Luis Antonio Arguello Escutia 

Este proyecto es una aplicación web moderna y 
minimalista desarrollada para el Hotel Quinta Dalam, 
ubicado en Coeneo, la capital mundial del Jaripeo en Michoacán.

Tecnologías Utilizadas

* **React** - Librería para interfaces de usuario
* **Vite** - Herramienta de construcción y entorno de desarrollo rápido
* **CSS / HTML** - Diseño y maquetación web

El código está organizado de manera modular para mantener las buenas prácticas:

* `/src/assets` - Imágenes, logos y recursos multimedia.
* `/src/components` - Componentes reutilizables de React (Navbar, Footer, etc.).
* `/src/pages` - Las vistas o pantallas completas de la página web.
* `/src/styles` - Hojas de estilo CSS.
